## рҹҗӣ PROBLEMAS COMUNES

### вқҢ No veo productos

```
SoluciГіn:
1. npm run dev
2. http://localhost:5173/admin
3. Clic "рҹ“Ұ Importar Productos Sego"
```

### вқҢ Scraper no funciona

```
SoluciГіn:
1. npm install puppeteer --force
2. Verificar credenciales en scraper-local.js
3. npm run scrape
```

### вқҢ Error 404 en Edge Function

```
SoluciГіn:
ВЎNo uses Edge Function!
Usa "Importar Productos" o "npm run scrape"
```
